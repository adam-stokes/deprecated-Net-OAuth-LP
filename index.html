Net-oauth-lp

Perl interface for Launchpad.net oauth authorization and client utilities

View project onGitHub

About

I like writing in perl every now and then so I put this together so I could access certain information within Launchpad.net via a perl bot I use.

If you want a more full featured client and don't mind python I highly suggest using launchpadlib.
